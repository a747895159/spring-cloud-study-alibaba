# 前端设计插件

生成独特的、生产级的前端界面，避免通用的 AI 美学风格。

## 功能介绍

Claude 会自动对前端工作使用此技能。创建具备以下特点的生产就绪代码：

- 大胆的美学选择
- 独特的排版和配色方案
- 高影响力的动画和视觉细节
- 上下文感知的实现

## 用法

```
"为音乐流媒体应用创建一个仪表盘"
"为 AI 安全初创公司构建一个落地页"
"设计一个带暗色模式的设置面板"
```

Claude 会选择一个明确的美学方向，并以精心打磨的细节实现生产代码。
